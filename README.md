# SNMP_TCS34725_RBG_Sensor
Establishing SNMP on dumb devices, by observing the LEDs on dumb devices.

To connect to the Raspberry Pi please connect the pins as follows:
SDA       P3 GPIO 0 (SDA)
SCL       P5 GPIO 1 (SCL)
3V3       P1 3V3
GND       P6
Optional:
LED       P9 (Disables LED)
